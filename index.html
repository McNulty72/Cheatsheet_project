Lists Tags

Tag Name Description
<ol> Ordered List Represents an ordered list of items.
<ul> Unordered List Represents an unordered list of items.
<li> List Item Represents an item in a list.

CSS Selectors

Selectors

Selector Name Specifity Weight Example Description
* Universal Selector 0,0,0,0 * { color: red; } Selects all elements and applies the specified styles
element Element Selector 0,0,0,1 p { font-size: 16px; } Selects all <p> and sets the font size to 16 pixels
.class Class Selector 0,0,1,0 .higlight { background-color: yellow; } Selects elements with the class "higlight" and sets the background color to yellow
#id ID Selector 0,1,0,0 #logo { width: 200px } Selects the element with the id "logo" and sets the width to 200 pixels
[atrribute] Attribute Selector 0,0,1,0 [type="submit"] { background-color: blue; } Selects elements with the attribute type set to "submit" and sets the background color to blue
:pseudo Pseudo-Class Selector 0,0,1,0 a:hover { color: purple; } Selects <a> elements when they are being hovered over and sets the color to purple
::element Pseudo-Element Selector 0,0,1,0 p::first-line { font-weight: bold; } Selects the first line of <p> elements and set the font weight to bold
+ Adjacent Sibling Selector 0,0,1,0 h2 + p { margin-top: 10px } Selects <p> elements that directly follow an <h2> element and sets the top margin to 10 pixels
> Child Selector 0,0,1,0 ul > li {list-style: none; } Selects <li> elements that are direct children of <ul> elements and removes the bullet points
Descendant Selector 0,0,1,0 div span { color: green; } Selects <span> elements that are descendants of <div> elements and sets the color to green
~ General Sibling Selector 0,0,1,0 h3 ~ p { font-style:italic; } Selects <p>elements that are siblings of an <h3> element and applied italic font style
style="" Inline Style 1,0,0,0 <p style="color: blue;">This is a paragraph.</p> Applies the inline style directly to the <p> element, setting the color to blue
!important Important ∞ h1 { color: red !important; } Sets the color of all <h1> elements to red, overriding any other specificity rules
